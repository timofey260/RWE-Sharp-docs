# RWE# Docs

RWE# API and Modding docs, made with [Sphinx](https://www.sphinx-doc.org/en/master/)

# Building

Initiate [Virtual environment](https://docs.python.org/3/library/venv.html#creating-virtual-environments) and install required packages

[GraphViz](https://graphviz.org/) is recommended but not required

To build website, use:
```shell
make html
```

Make sure RWE#'s repo is in the same folder as this one(or change rwe#'s path in `config.py`)
